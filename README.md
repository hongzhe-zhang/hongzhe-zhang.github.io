# Hongzhe Zhang Academic Website

This is a lightweight static academic homepage designed for GitHub Pages.

## Files

- `index.html`: homepage content and section structure
- `styles.css`: responsive layout and visual styling
- `script.js`: mobile navigation behavior
- `assets/research-banner.png`: generated research-themed banner image

## GitHub Pages

Push these files to the root of a GitHub Pages repository, such as `your-github-username.github.io`.
Then enable Pages from the repository settings if it is not already enabled.
